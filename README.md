# Nuvi Academy Bot (@isroil_aibot) 🚀

Nuvi Academy loyihasi uchun maxsus ishlab chiqilgan Telegram bot va Mini App tizimi.

## 🌟 Imkoniyatlar

- **Aiogram 3**: Zamonaviy va tezkor bot asosi.
- **Lokalizatsiya**: O'zbek tilidagi to'liq interfeys.
- **Funnel & CRM**: Mijozlarni ro'yxatdan o'tkazish va segmentatsiya qilish.
- **Referal Tizimi**: Takliflar orqali foydalanuvchilarni jalb qilish.
- **Obuna va To'lovlar**: Telegram Invoices orqali to'lovlarni qabul qilish.
- **Admin Panel**: Kurslar, qo'llanmalar va foydalanuvchilarni boshqarish uchun qulay dashboard.

## 🛠 Texnologiyalar

- **Python 3.12**
- **Aiogram 3**
- **FastAPI** (Webhook va API uchun)
- **SQLAlchemy** (Ma'lumotlar bazasi)
- **React/Vite** (Admin Dashboard)

## 🚀 Ishga tushirish

1. `.env` faylini sozlang (Bot token va boshqa kalitlar).
2. Kutubxonalarni o'rnating: `pip install -r requirements.txt`.
3. Botni ishga tushiring: `python -m bot.main`.
4. API va Webhookni ishga tushiring: `python -m uvicorn api.main:app`.
